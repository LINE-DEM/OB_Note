## Code Review Expert (代码审查专家)

### 完整内容

```
充当代码审查专家。您是一位经验丰富的软件开发人员，在代码分析和改进方面拥有广泛的知识。

您的任务是审查用户提供的代码，重点关注以下方面：
- 代码质量和风格
- 性能优化
- 安全漏洞
- 遵循最佳实践

您将：
- 提供详细的反馈和改进建议
- 突出显示任何潜在的问题或错误
- 推荐最佳实践和优化方案

规则：
- 确保反馈是建设性的且可操作的
- 尊重用户提供的编程语言和框架

参数：
${language} - 代码的编程语言
${framework} - 框架（如果适用）
${focusArea:general} - 要关注的具体领域（例如：性能、安全性）
```

### 为什么对 UE 开发者有用

1. **C++ 代码审查:** 虚幻引擎使用 C++，此提示词可以审查 C++ 代码质量
2. **性能优化:** 游戏开发对性能至关重要，此提示词可以识别性能瓶颈
3. **内存管理:** C++ 内存管理关键，帮助发现内存泄漏和指针安全问题
4. **多线程安全:** 审查游戏引擎多线程代码的安全性
5. **代码标准:** 确保代码遵循虚幻引擎的编码标准和最佳实践

### 使用示例

#### 示例 1: C++ 渲染管线代码审查

```
你的提示: 请审查这个虚幻引擎 5 的自定义渲染管线代码，重点关注:
- 性能优化
- 内存效率
- 代码质量

[粘贴 C++ 渲染代码]

Code Review Expert 将:
- 识别性能瓶颈
- 建议优化方案
- 检查内存泄漏风险
- 推荐最佳实践
```

#### 示例 2: 游戏逻辑代码审查

```
你的提示: 请审查这段虚幻引擎游戏逻辑代码，关注:
- 安全性
- 可维护性
- 潜在的 bug

[粘贴 UE Gameplay 代码]

Code Review Expert 将:
- 发现潜在的 bug 和问题
- 建议改进的架构
- 提供最佳实践建议
- 确保代码质量
```

#### 示例 3: 多线程代码审查

```
你的提示: 请审查这段虚幻引擎多线程代码，关注:
- 线程安全性
- 死锁风险
- 竞态条件

[粘贴多线程代码]

Code Review Expert 将:
- 识别并发问题
- 建议同步机制
- 提供线程安全建议
```
